# Python Backend Labs

Hands-on Python backend experiments and mini-projects focused on APIs, data handling, and real-world backend development.

## Goals
- Practice backend engineering fundamentals
- Build APIs and backend utilities in Python
- Learn clean project structure and best practices

## Tech Stack
- Python 3
- Django / FastAPI / Flask (where applicable)
- SQLite / PostgreSQL
- Requests, SQLAlchemy

## Structure
All experiments and source code are organized inside the `src/` directory with examples and tests.
